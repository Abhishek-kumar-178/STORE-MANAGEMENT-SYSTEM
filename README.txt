Store Management System

A simple database project built with MySQL to manage a retail store's inventory, staff, and sales.

Files:
- project_store_items.sql   - product catalog with pricing and stock levels
- project_staffs.sql        - staff records
- project_sales_report.sql  - sales data and reporting queries
- project_ordered_stocks.sql - stock order tracking

Requirements:
- MySQL 8.0 or above

Setup:
Import each .sql file into your MySQL instance in the order listed above.
The database is named `project`.

Notes:
The dataset includes ~20 products across categories like Laptops, Smartphones, and Accessories.
Prices are in INR.
